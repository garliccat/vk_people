# Сбор открытых данных пользователей VK путём банального перебора UID

## Используется API VK и сервисный ключ доступа. (можно и с ключом доступа пользователя)

### Собирает:

- Имя
- Фамилию
- Пол
- Страна
- Город 
- Дата рождения
- Любимая музыка
- Любимые фильмы
- Семейный статус (отношения)

Первоначально сбор планировался только пары сотен тысяч, но в итоге собирается всё батчами по 200 пользователей (дабы не было исключений из-за длины строки передаваемой requests), и лимит был установлен в 10000000.
